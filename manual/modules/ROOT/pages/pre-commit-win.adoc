=  Pre-commit on windows example

== General steps
One route known to occasionally work:

. Install python as administrator on the command line using chocolatey:
   `choco install python`
. Start _Windows Settings_ | _Apps_.
. Scroll to the _Python_ app, click _Modify_
. In python app settings, click _Modify_.
. Check the _Py Launcher checkbox_, click _Next_
. Click _Install_.
. Install pre-commit on the command line using `py -m pip install pre-commit`.
  This often creates warnings about missing PATH; these can be ignored.
. On the command line in the plugin directory activate pre-commit in the
  project using `py -m pre_commit install`
