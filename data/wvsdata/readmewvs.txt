The 2 files in thiis zip, wvs43.dat and wvs1.dat are World Vector Shoreline (WVS) map 
files used by the WXTide32 program.  These files are open source and are a subset of 
the full set of data which totals over 40 MB in zip form.

See http://wxtide32.com for more information on the program and using these map files.
